概要Overview

kotoji が提供するもの。 What kotoji gives you.

他のどこも一か所では提供しない、この組み合わせ — セルフホスト + MCP ネイティブな AI アクセス + 唯一の真実としての git + 非エンジニアにやさしいエディタ。 The one combination nothing else offers in a single place: self-hosting + MCP-native AI access + git-as-source-of-truth + a non-engineer-friendly editor.

主な機能Core features

アップロード & 配信Upload & serve

静的ファイルの .zip をドロップすれば即座に URL が手に入ります。どんなアセットパスの書き方でも動きます。 Drop a .zip of static files and get an instant URL — any asset-path style works.

プロジェクト別サブドメインPer-project subdomains

各ツールは your-tool.example.com に住み、独自のオリジンで隔離されます。 Each tool lives at your-tool.example.com, isolated on its own origin.

ブラウザ内編集In-browser editing

差分ビュー付きの Monaco エディタでさっと修正 — チェックアウトもローカルのツールチェーンも不要です。 A Monaco editor with diff view for quick fixes — no checkout, no local toolchain.

MCP ネイティブMCP-native

Claude（または任意の MCP クライアント）を接続して、サイトを直接 list / read / write / publish できます。 Connect Claude (or any MCP client) to list / read / write / publish your sites directly.

ユーザー単位トークンPer-user tokens

1 つのトークンはあなたに属し、メンバーである全プロジェクトにまたがります。実効スコープ = トークン ∩ ロール。 One token belongs to you and spans every project you're a member of; effective scope = token ∩ role.

git が真実git is the truth

すべての保存がコミットです。履歴・差分・ロールバックが無料で付いてきます。 Every save is a commit. History, diff and rollback come for free.

下書き → 公開Draft → published

ブランチ上で安全に作業し、ワンアクションで本番へ昇格。各ブランチにプレビュー URL が付きます。 Work safely on branches; promote to production in one action. Each branch gets a preview URL.

公開範囲のティアVisibility tiers

サイトごとに Public・Members・Private — 既定は Private。 Public, Members or Private per site — Private by default.

GitHub ミラー & TLSGitHub mirror & TLS

公開のたびの任意の外部バックアップと、自動ワイルドカード HTTPS のためのオプトイン Traefik オーバーレイ。 Optional off-box backup on every publish, and an opt-in Traefik overlay for automatic wildcard HTTPS.

なぜセルフホスト?Why self-host?

  • SaaS のビルダー（Val Town、v0、Bolt、Lovable）はプロプライエタリで、セルフホストできません。 SaaS builders (Val Town, v0, Bolt, Lovable) are proprietary and can't be self-hosted.
  • セルフホストの PaaS（Coolify、Dokploy）は git push の CI フローを前提とします — ブラウザ内エディタも AI 統合もありません。 Self-hosted PaaS (Coolify, Dokploy) assume a git-push CI flow — no in-browser editor, no AI integration.
  • kotoji はこれら 4 つの要素を、あなた自身のサーバー上で組み合わせます。 kotoji combines all four pieces on your own server.
